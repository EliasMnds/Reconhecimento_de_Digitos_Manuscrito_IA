O reconhecimento de dígitos manuscritos é uma tarefa comum de reconhecimento de imagem em que os computadores reconhecem dígitos em imagens manuscritas. Diferentemente das fontes impressas, a caligrafia de pessoas diferentes tem estilos e tamanhos diferentes, dificultando o reconhecimento de caligrafia por computadores. Este notebook usa ferramentas de aprendizado profundo e TensorFlow para treinar um modelo usando conjuntos de dados MNIST.

- Leitura do conjunto de dados de dígitos manuscritos MNIST.

- Implementação da regressão softmax usando APIs de alto nível.
